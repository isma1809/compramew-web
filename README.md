# Compramew Web

Sitio web de soporte para la aplicación móvil Compramew.

## Contenido

- **Universal Links**: Configuración de Apple App Site Association
- **Auth Callback**: Página de confirmación de email

## URLs

- Production: https://compramew.es
- Apple App Site Association: https://compramew.es/.well-known/apple-app-site-association
- Email Callback: https://compramew.es/auth/callback

## Deployment

GitHub Pages deployment automático desde la rama `main`.
